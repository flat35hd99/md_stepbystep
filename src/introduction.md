# イントロダクション

## ライセンス表記

このウェブページでは、学部4年で計算機科学の研究室に配属され、タンパク質などの分子動力学シミュレーションを行うことになった人、または、それに相当する人を対象としたコンテンツを提供します。

このウェブページで配信している文書・画像はMITライセンスで配布します。ただし、アイコンなどの一部コンテンツについてはmdbookによるライセンス下にあるため、取り扱いに注意してください。再利用可能かどうかの判断に困ったら、リポジトリの[src](https://github.com/flat35hd99/md_stepbystep/tree/main/src)ディレクトリ以下のものは間違いなくMITライセンスで配布されているものなので、ここにあるかどうかで判断してください。

このコンテンツは、markdownで記述したドキュメントを[mdbook](https://github.com/rust-lang/mdBook)でビルドし、[actions-gh-pages](https://github.com/peaceiris/actions-gh-pages)でgithub pagesにデプロイし、配信しています。
